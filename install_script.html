LstSim Commander

    lstsim.de Userscript installieren

    Um den lstsim-commander mit lstsim.de zu verbinden ist eine Browsererweiterung notwendig.

    1. Tampermonkey-Browsererweiterung installieren
      Links zum Installieren für die gängigsten Webbrowser:
      • Google Chrome
      • Microsoft Edge
      • Mozilla Firefox
      • Apple Safari
    2. Sobald die Browsererweiterung Tampermonkey installiert ist auf diesen Link klicken um das Script für lstsim-commander zu installieren.
    3. Nun sollte sich eine Seite mit dem Tampermonkey-Logo öffnen, auf dieser Seite den Button "Installieren" klicken.
    4. Nun kann der lstsim.de gestartet werden.
